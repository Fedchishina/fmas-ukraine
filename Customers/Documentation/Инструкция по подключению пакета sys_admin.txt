Для подключения необходимо:
1. Найти файлы: sys_admin.bpl, sys_admin.dcp и положить их рядом со своим екзешником. Должны лежать в релизе в корне с приложениями.
2. Подключить пакет sys_admin к exe-шнику. Project->Options->Packages, в поле Runtime packages дописать sys_admin.
3. В uses главной формы дописать SysAdmin.
4. В onShow главной формы вставить:  InitializaAdminSystem(Self, Database.Handle, ID_SYSTEM, ID_USER, isSuperAdmin);
    ID_SYSTEM    - идентификатор вашей системы. Находиться в таблице SYS_SYSTEMS. Если его там нет, то необходимо добавить.
    ID_USER      - идентификатор пользователя который залогинелся в вашу программу.
    isSuperAdmin - Признак супер админа. Пока должен быть false.
5. В onClose главной формы вставить: FinalizeAdminSystem;
   Можно вставить и в onDestroy, только коннект с бызой должен быть.
6. Чтобы показать форму с пользователями необходимо вызвать: AdminShowUsers(Self);
7. Чтобы показать форму с ошибками необходимо вызвать: AdminShowErrors(Self);

Примечания:
1. Запрос о желании пользователя выйти из программы должен быть в onCloseQuery.
2. При принудительном завершении программы событие onCloseQuery игнорируется.
3. Мой пакет переопределяет событие onException объекта application.
